Mesquite-o-tastic: A Mesquite bridge to the Phylotastic web service

Peter E. Midford and the Phylotastic Project

June 2012

This package presently contains a single module that allows a user to retrieve a tree from phylotastic from within mesquite. All that is required is a character matrix and that the associated taxon block be completely filled in. The module currently operates as though it were an inferencing tree source though this is not actually the case (and this should be fixed in the future).
